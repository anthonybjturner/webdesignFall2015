What I know about HTML tags

The HTML tags that we've looked at so far include:

html
This tag opens and closes the document.
body
This tag starts and ends the main body of the document.
head
This tag starts and ends the head of the document. It also contains the title.
title
This tag contains the title of the page, which will be displayed on the browser tab.
p
This is a paragraph.
ul
This is an unordered list, which consist of bullet points.
li
this is a list element, used in both ordered and unordered lists.
ol
This is an ordered list, which is numbered.
em
This tag specifies that some sort of emphasis should be applied to the text, which can vary depending on browsers.
h1
This is a top level header
h2
This is a second level header
table
This starts a table
thead
This is the header section of a table
tbody
This is the body section of a table
tr
This is a table row, it's used in both header and body sections of a table
th
This is a header table cell
td
This is header body cell
dl
This starts a description list
dt
This is a term in a description list
dd
This is a description or definition in a description list
